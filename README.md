IndiGo Airline Data Analysis Project

Project Overview

This project focuses on exploratory data analysis (EDA) of IndiGo Airlines flight data to uncover insights related to flight operations, pricing trends, routes, delays, and performance metrics. The goal is to demonstrate data cleaning, analysis, and visualization skills using real-world airline data.

Objectives

Analyze flight routes and frequency

Study ticket price patterns across routes and time

Identify trends in delays and cancellations

Understand demand patterns by source, destination, and time

Generate actionable insights using data visualization

Tools & Technologies

Programming Language: Python

Libraries:

Pandas

NumPy

Matplotlib

Seaborn

Environment: Jupyter Notebook

Dataset Description

The dataset contains flight-level information for IndiGo Airlines, including:

Airline Name

Source City

Destination City

Departure & Arrival Time

Flight Duration

Ticket Price

Date of Travel

Dataset is for educational and analytical purposes.

Key Analysis Performed

Data cleaning and preprocessing

Handling missing and inconsistent values

Exploratory Data Analysis (EDA)

Route-wise and city-wise analysis

Price distribution and trend analysis

Visualizations for better interpretation

Visualizations

The project includes:

Bar charts for route frequency

Line charts for price trends

Box plots for ticket price distribution

Heatmaps for correlation analysis

Insights & Findings

Certain metro routes show consistently higher ticket prices

Early morning and late-night flights tend to be cheaper

High-demand routes have more frequent flights

Ticket prices vary significantly based on travel date and route
